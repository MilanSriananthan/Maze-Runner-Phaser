# Maze Runner

## Purpose

Used Phaser 3.0 technology and prior knowledge to create a simple game to demonstrate my quick
understanding of this technology.

## Functions

- Player Movement
- Game Logic
- Repeated tries
- Game sounds
- Game Images
- Web and mobile responsive
- Level Creation
